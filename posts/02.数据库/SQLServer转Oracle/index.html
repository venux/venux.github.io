1. 情景

2. 工具

  • SQL Server 自带的 bcm（注意：需将 SQL Server 安装目录下对应版本的 Tools\Binn 目录，如C:\Program Files\Microsoft SQL Server\100\Tools\Binn 添加到环境变量 Path 中）；
  • SQL Developer 17.2(注意：需安装配置 Java 8)；
